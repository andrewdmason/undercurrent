# AI Prompts

This folder contains markdown files for AI prompts used throughout the app.

## Prompts

- `onboarding-agent.md` - System prompt for the onboarding chatbot that creates strategy docs
- `idea-generation.md` - Prompt for generating new video ideas
- `underlord-prompt.md` - Prompt for creating Descript Underlord prompts from ideas

These files make it easy to iterate on prompts without changing code.
